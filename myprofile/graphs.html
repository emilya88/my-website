About This IoT Project

Using an ESP32 microcontroller, various environmental sensors, and Amazon cloud services, we built a system capable of reading real-time air-quality parameters, transmitting data wirelessly, and storing it for later analysis and visualization.

The context of the project is strongly linked to the increasing concern regarding Armenia's air quality, where pollution levels have become a noticeable health and environmental issue. This provided both motivation and real-world relevance to the technical work.
